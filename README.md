# 🚀 Sumit Dwivedi | 3D Portfolio

A premium, high-performance 3D portfolio website built for **Sumit Dwivedi**, an aspiring AI Engineer and Full Stack Developer. This project features immersive 3D visuals using React Three Fiber, a robust MERN backend, and automated email notifications.

![Portfolio Preview](https://images.unsplash.com/photo-1507238691740-187a5b1d37b8?auto=format&fit=crop&q=80&w=1200)

## ✨ Features
- **Immersive 3D Experience**: Interactive 3D Hero section and dynamic starfield background.
- **Glassmorphism UI**: Modern, sleek design with futuristic neon accents.
- **Full-Stack Contact Form**: Integrated with MongoDB Atlas for data storage and Nodemailer for instant email alerts.
- **Dynamic Projects Showcase**: Highlighting key projects like the Elite Exam Portal and Trading Bot.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop experiences.
- **Internship Tracking**: Dedicated section for ongoing Data Science internship details.

## 🛠️ Tech Stack
- **Frontend**: React.js, TypeScript, Vite, Framer Motion, Lucide Icons.
- **3D Graphics**: Three.js, React Three Fiber, @react-three/drei.
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB Atlas (Cloud).
- **Email Service**: Nodemailer (Gmail).
- **Styling**: Vanilla CSS (Custom Design System).

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- MongoDB account (for cloud DB)
- Gmail App Password (for email notifications)

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/sumitdwivedi681-ops/Portfolio---SumitDwivedi.git
   cd Portfolio---SumitDwivedi
   ```

2. **Setup Backend:**
   ```bash
   cd backend
   npm install
   ```
   Create a `.env` file in the `backend` folder:
   ```env
   PORT=5000
   MONGODB_URI=your_mongodb_atlas_uri
   EMAIL_USER=your_email@gmail.com
   EMAIL_PASS=your_gmail_app_password
   ```
   Start the server: `npm start`

3. **Setup Frontend:**
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```
